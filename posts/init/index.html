Init

Create repository for GitHub pages

$ cd ~/src/github.com/drillbits
$ mkdir drillbits.github.io
$ cd drillbits.github.io
$ git init
$ hub create drillbits.github.io

Install hugo

$ go get github.com/spf13/hugo
$ ghq get https://github.com/spf13/hugoThemes.git

Create site

$ cd ~/src/github.com/drillbits
$ hugo new site drillbits.jp

Write initial content

$ cd drillbits.jp
$ hugo new posts/init.md
$ vim content/posts/init.md
(write something)

Preview

$ mkdir themes
$ cp -r ~/src/github.com/spf13/hugoThemes/simple-a themes/
$ rm -rf themes/simple-a/.git
$ hugo server --watch -D -t simple-a
(snip)

Create private repository

$ rm -rf public
$ git init
$ hub create -p drillbits.jp
$ touch archetypes/.gitkeep
$ touch data/.gitkeep
$ touch layouts/.gitkeep
$ touch static/.gitkeep
$ git add .
$ git commit -m 'Initial commit'
$ git push origin master

Deploy

$ git submodule add git@github.com:drillbits/drillbits.github.io.git public
$ hugo -t simple-a
$ cd public
$ git add -A
$ git commit -m 'Build'
$ git push origin master
$ cd ..
